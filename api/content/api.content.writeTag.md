## 接口描述
用户给内容贴标签
## API Path
/api/content/writeTag
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
|tag    |string    |是    |标签内容    |
|uid    |Int    |是    |发布标签的人    |
|cid    |Int    |是    |内容id    |
|customerId    |Int    |是    |被贴标签的人（废弃）    |
|type    |Int    |是    |贴标类型 1原生内容，2老徐文章 3 活动    |
## 返回报文
    {
    "code": 20044,
    "message": "标签发布成功"
    }
