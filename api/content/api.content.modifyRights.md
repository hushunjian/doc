## 接口描述
用户修改内容权限
## API Path
/api/content/modifyRights
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
|rights    |Int    |是    |0仅自己1公开    |
|cid    |Int    |是    |内容id    |

    "message": "内容权限修改成功"
    }
