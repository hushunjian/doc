## 接口描述
设置密码
## API Path
/api/user/modifyEncrypt
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
|userName    |string    |是    |用户名（手机号）    |
|encrypt    |string    |是    |老密码    |
## 返回报文
    {
        "accessToken": "48c3847d11754f708628c70211b17b24",
        "code": 2002,

    }
