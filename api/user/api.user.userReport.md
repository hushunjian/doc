## 接口描述
举报接口
## API Path
/api/user/userReport
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
|uid    |int    |是    |举报人    |
|cid    |int    |是    |内容id    |
|reason    |int    |是    |举报理由（1广告 、2色情、3其他）    |
|attachment    |String    |可选    |其他原因，用户填写    |
## 返回报文
    {
    "code": 20054,
    "message": "举报成功"
    }
