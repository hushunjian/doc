## 接口描述
将用户的米汤币充值到王国中
## API Path
/api/user/rechargeToKingdom
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
|topicId    |int    |是    |王国ID    |
|Coin    |int    |是    |充值数量    |
## 返回报文
    {
    "code": 200,
    "message": "操作成功"
    }
