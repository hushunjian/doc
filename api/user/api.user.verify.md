## 接口描述
验证码发送和校验
## API Path
/api/user/verify
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
|action    |string    |是    |在发送验证码时传 0 (忘记密码 2 , 校验时该参数传1,验证码注册 传5,验证码登录时传 4)    |
|mobile    |string    |是    |用户手机号码    |
|channelAdapter    |string    |是    |通道 1短信2语音    |
|verifyCode    |String    |否    |校验时传    |
## 返回报文
    {
    "code": 2004,
    "message": "验证码发送成功"
    }
