## 接口描述
获取粉丝列表（按关注时间排序）
## API Path
/api/user/showFans2
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
|customerId    |Int    |是    |用户uid    |
|sinceId    |Int    |是    |分页标识，按页数分页，第几页传几    |
