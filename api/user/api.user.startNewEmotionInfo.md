## 接口描述
最近一次情绪信息查询接口
## API Path
/api/user/startNewEmotionInfo
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
|image    |string    |是    |情绪总结信息图片七牛key    |
|w    |int    |是    |情绪总结信息图片宽度    |
|h    |int    |是    |情绪总结信息图片高度    |
|source    |int    |是    |来源，同王国创建接口    |
|    |    |    |    |
## 返回报文
    {
    "code": 200,
    "message": "操作成功"
    }
