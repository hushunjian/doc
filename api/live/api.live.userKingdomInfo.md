## 接口描述
获取用户王国相关信息接口，比如已经创建了几个王国等
## API Path
/api/live/userKingdomInfo
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
## 返回报文
    参数名称	类型	备注
    totalKingdomCount	int	总王国数
    validKingdomCount	int	有效王国数（未删除的）
