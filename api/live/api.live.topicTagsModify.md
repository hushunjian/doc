## 接口描述
王国标签更新接口
## API Path
/api/live/topicTagsModify
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
|topicId    |long    |是    |王国ID    |
|tags    |string    |否    |王国标签，多个之间以英文分号分隔    |
## 返回报文
    {
    "code": 200,
    "message": "操作成功"
    }
