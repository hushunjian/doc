## 接口描述
从王国列表中移除王国，此王国有新内容的时候会再次出现。
## API Path
/api/live/removeKingdom
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
|topicId    |long    |是    |王国ID    |
## 返回报文
    {
    "code": 200,
    "message": "操作成功"
    }
    or
    {
    "code": 50066,
    "message": "你无权操作"
    }
