## 接口描述
在王国互动页面上置顶操作，并且仅影响该页面的王国列表排序
## API Path
/api/live/myTopicOpt
## 请求报文
|名称         |类型           |是否必须   |备注                                 |
|-------------|:--------------|:---------:|:------------------------------------|
|topicId    |long    |是    |王国    |
|action    |int    |是    |1置顶    |
|2取消置顶    |    |    |    |
## 返回报文
    {
    "code": 200,
    "message": "操作成功"
    }
