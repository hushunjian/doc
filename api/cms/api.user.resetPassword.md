## 接口描述
更新用户密码
## API Path
/api/user/resetPassword
## 请求说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|token     |string  |权限token        |
|id |in  |用户id           |
## 请求报文
    {
        "token":"{{token}}",
        "id": 1
    }
## 响应说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|status   |int     |2000:成功        |
## 返回报文
	{
		"status":2000	
	}
