## 接口描述
设置用户邀请状态(是 或否)
## API Path
/api/appuser/setUserInviteStatus
## 请求说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|token    |string  |权限token        |
|uid      |用户ID  |用户ID           |
|isInvited|int     |是否运营邀请(0:是；1：否)  |

## 请求报文
    { 
      "token":"99999999-9999-9999-9999-999999999999",
      "uid": 1234,
      "isInvited":1
    }
    
## 响应说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|status   |int     |2000:成功        |
## 返回报文
  {
    "status":2000 
  }
