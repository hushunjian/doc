## 接口描述
删除操作(从榜单列表中删除王国或榜单,用户信息)
## API Path
/api/listRank/deleteEntry
## 请求说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|token    |string  |权限token        |
|id       |int     |榜单记录ID(王国/榜单/用户) |


## 请求报文
    { 
      "token":"99999999-9999-9999-9999-999999999999",
      "id":"2"
    }
## 响应说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|status   |int     |2000:成功        |
## 返回报文
  {
    "status":2000 
  }
