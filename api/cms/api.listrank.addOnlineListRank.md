## 接口描述
添加上线榜单
## API Path
/api/listRank/addOnlineListRank
## 请求说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|token    |string  |权限token        |
|id       |int     |榜单ID           |
|type     |int     |上线榜单类型(1:用户；2：王国) |
|sort     |int     |榜单排序         |
## 请求报文
    { 
      "token":"99999999-9999-9999-9999-999999999999",
      "id": 123,
      "type": 1,
      "sort": 5
    }
    
## 响应说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|status   |int     |2000:成功        |
## 返回报文
  {
    "status":2000 
  }
