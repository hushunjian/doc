## 接口描述
设置上市王国挂牌交易
## API Path
/api/kingdom/kingdomTransaction
## 请求说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|token    |string  |权限token        |
|id       |int     |上市王国id           |
|meNumber |int     |新国王米汤号     |

## 请求报文
    { 
      "token":"99999999-9999-9999-9999-999999999999",
      "id": 1245,
      "meNumber": 312231
    }
    
## 响应说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|status   |int     |2000:成功        |
## 返回报文
  {
    "status":2000 
  }
