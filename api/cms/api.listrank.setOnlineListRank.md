## 接口描述
已选榜单上下架处理
## API Path
/api/listRank/setOnlineListRank
## 请求说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|token    |string  |权限token        |
|id       |int     |榜单ID           |
|status   |int  |状态(0:下架; 1:上架)      |
## 请求报文
    { 
      "token":"99999999-9999-9999-9999-999999999999",
      "id": 123,
      "status": 0
    }
    
## 响应说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|status   |int     |2000:成功        |
## 返回报文
  {
    "status":2000 
  }
