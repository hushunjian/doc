## 接口描述
设置用户状态(正常 或 封号)
## API Path
/api/appuser/setUserStatus
## 请求说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|token    |string  |权限token        |
|uid      |用户ID  |用户ID           |
|status   |int     |状态（0：正常；1：封号)    |

## 请求报文
    { 
      "token":"99999999-9999-9999-9999-999999999999",
      "uid": 1234,
      "status":1
    }
    
## 响应说明
|参数名   |类型    |备注             |
|---------|--------|-----------------|
|status   |int     |2000:成功        |
## 返回报文
  {
    "status":2000 
  }
